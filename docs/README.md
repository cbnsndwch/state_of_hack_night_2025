# State of the Hack Night 2025 - Project Plan

## Project Overview

**Goal**: Create a static, interactive "Year in Review" website for the Miami Hack Night community.
**Target Audience**: Community members, partners, and sponsors.
**Format**: Single-page application (SPA) deployed as a static site (SSG).

## Tech Stack

- **Build Tool**: [Vite](https://vitejs.dev/)
- **Framework**: [React](https://react.dev/) (TypeScript)
- **Routing**: [React Router 7](https://reactrouter.com/) (latest)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Design Theme**: Monochromatic Green (Luma/Hacker aesthetic), Dark Mode.
- **Visualization**:
    - Charts: [Recharts](https://recharts.org/)
    - Maps: [Mapcn](https://mapcn.vercel.app/) (D3 + TopoJSON)
- **Deployment**: GitHub Pages / Static Hosting

## Data Source

The project utilizes pre-fetched JSON data from the `luma-event-fetcher` project.

- **Source Config**: No runtime API calls. Data is imported at build time.
- **Key Files**:
    - `events-complete-*.json`: Full event and guest lists.
    - `zip-counts.json`: Aggregated zip code data for the map.
    - `question-groups.json`: Aggregated survey answers (roles, interests).

## üìù Implementation Hints for Agents

**Data Loading:**
Since this is an SSG/SPA, simply import the JSON files directly in the code:

```typescript
import events from '@/data/events-complete.json';
import zipCounts from '@/data/zip-counts.json';
```

_Note: Ensure `tsconfig.json` allows `resolveJsonModule`._

**Map Visualization:**
The `zip-counts.json` file contains `{ "33139": 45, "33130": 120 }`.
The `zip-geo-cache.json` contains lat/lon for zips.
Merge these in `CommunityMap.tsx`. Instead of tiles, use `mapcn` (D3) to project these coordinates onto an SVG map (likely focusing on FL or Miami-Dade).

**Styling Theme:**

- Primary: Miami Neon Cyberpunk/Vaporwave accents
- Dark Mode is recommended for "Hack Night" vibes.
